# papershift-cli

```sh
$ papershift-cli
$ Build summary...
$ Started at 07:36 worked for 7:44:47
```

## Installation

```js
npm install -g papershift-cli
```

Add the following environment variables to your profile

```sh
PAPERSHIFT_USER="user_id"
PAPERSHIFT_TOKEN="token"
```

## Usage

```sh
papershift-cli
```
